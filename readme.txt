== Description ==
The Wordpress Animated Login Page plugin is a customizable plugin that enhances the appearance of the WordPress login page by adding animated elements. It allows you to replace the default login form with a visually appealing and engaging interface. This plugin is useful for website owners who want to provide a unique login experience to their users.

Features

    Customizes the WordPress login page with animated elements.
    Removes the default CSS styles from the login page to provide a clean canvas for customization.
    Adds custom CSS styles to the login page for enhanced visual appeal.
    Modifies the HTML structure of the login, registration, and lost password forms to provide a customized user interface.
    Includes custom JavaScript files to enable animations and interactive elements on the login page.

Installation

    Download the plugin ZIP file from the WordPress Animated Login Page Plugin page.
    Log in to your WordPress admin dashboard.
    Navigate to "Plugins" > "Add New."
    Click on the "Upload Plugin" button at the top.
    Choose the downloaded ZIP file and click "Install Now."
    Once the installation is complete, click "Activate Plugin."

Usage

After activating the plugin, the WordPress login page will be customized with animated elements. Users will see an updated login form, registration form, and lost password form with visually appealing styles and interactive features.
Customization

If you want to customize the appearance of the login page further, you can modify the CSS styles in the css/login-style.css file located in the plugin directory. You can add your own styles or modify existing ones to match your website's design.
Adding Additional Functionality

If you wish to add additional functionality or modify the behavior of the login page, you can edit the plugin's PHP files. Be cautious while making changes to ensure compatibility and prevent any errors.
Compatibility

The Wordpress Animated Login Page plugin is compatible with WordPress versions 4.0 and above.
